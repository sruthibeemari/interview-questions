HTML Interview Questions

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It structures web content by using a system of elements or tags that define the different parts of a webpage, such as text, images, links, and multimedia.

HTML creates the basic structure for a website, CSS adds style to that structure, and JavaScript makes it interactive and more functionally complex.

Doctype is the first line of an HTML document which tells the version of HTML that we are using.

Tags are used to define the beginning and end of an element, while Elements consist of the opening tag, content, and closing tag.

HTML entities are reserved characters that have special meanings when used in an HTML document. Each of these codes starts with an ampersand and ends with a semicolon.

HTML 5 improves upon HTML 4 by adding more features like multimedia support, a canvas for drawing, storage for web apps to work offline, and a way to find a user's location.

Start tag: Marks the beginning of the element's content. For example, the start tag for a paragraph is <p>
Content: The content of the element, which can be text or other elements.
End tag: Marks the end of the element's content. The end tag is the same as the start tag, but with a forward slash (/) after the first angle bracket. For example, the end tag for a paragraph is </p>.

We can use a <div> tag with width and height for making an empty square box without any content.

Attr refers to attributes in HTML. An attribute provides additional information about a particular tag. Some common attributes are:
  • Source attribute (src)
  • Hyperlink reference attribute (href)
  • Alternate (alt)
  • Target
  • Language (lang)

  • Title
  • Style
  • Meta
  • Link
  • Script

Meta tags are used in HTML at the head of the page, making them visible only to search engines and those who know how to view website source code. Metadata helps search engines understand the data on a website, aiding in content ranking on popular search engines like Google.

SEO stands for 'Search Engine Optimization', which is the process of getting traffic from free, organic, editorial, or natural search results in search engines. It aims to improve a website's position in search results pages (SERPs). Higher listings result in more visibility.

An <iframe> is a tag that allows a browser to display another document within the current page. Attributes like src, height, width, and others define and modify the region created by the <iframe> tag.

A form is used to collect detailed information. Form elements include:
  • Input
  • Label
  • Select
  • Legend
  • Fieldset
  • Option
  • Textarea

Semantic elements provide detailed information about the content. Examples include <article>, <header>, <footer>.
Non-semantic elements do not provide information about the content. Examples include <span>, <div>.

Relative file paths reference files located in the same folder of your website structure, while absolute file paths refer to external resources located elsewhere on the web.

In HTML, the "src" attribute specifies the source of an embedded resource like an image, while the "href" attribute defines the destination URL of a hyperlink, indicating where a link will take the user when clicked. "Src" is used to display content within a page, while "href" is used to navigate to a different page.
